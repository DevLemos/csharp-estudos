# Exercício 01: Registro de clientes

Você está desenvolvendo um sistema para um clube, onde a experiência do cliente é prioridade. 
Os membros comuns têm acesso básico, mas os clientes VIP possuem benefícios exclusivos, como 
níveis de fidelidade e identificadores personalizados. Seu desafio é criar uma estrutura que 
diferencie esses perfis.
